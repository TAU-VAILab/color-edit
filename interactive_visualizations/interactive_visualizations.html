Image Visualization Tool

This tool visualizes random samples from the CompColor benchmark, and results obtained over inital generated images and editing methods.
We report all metrics from the main paper, including CS and VQAScore for both the source and edited images,
as well as CDS with respect to both images, following the setup detailed in the supplementary material (CDS is shown only under the edited image)

  • Method: Select the editing method (edited images are shown on the right).
  • Model: Select the text-to-image generation model (inital images are shown on the left).
  • Color: Select the reference color.
